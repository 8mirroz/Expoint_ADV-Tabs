# Expoint Brand Spec

## Colors (OKLch)
Derived from `src/styles/tokens.css` (Vercel Light Theme / Stark Engineering).

- `--bg`:      oklch(99% 0.002 240); /* #fafafa */
- `--surface`: oklch(100% 0 0);      /* #ffffff */
- `--fg`:      oklch(18% 0.012 250); /* #171717 */
- `--muted`:   oklch(54% 0.012 250); /* #888888 */
- `--border`:  oklch(92% 0.005 250); /* #ebebeb */
- `--accent`:  oklch(58% 0.18 255); /* #0070f3 - Geist Blue */

## Typography
- **Display**: Inter, -apple-system, BlinkMacSystemFont, 'SF Pro Display', system-ui, sans-serif
- **Body**: Inter, -apple-system, BlinkMacSystemFont, 'SF Pro Text', system-ui, sans-serif
- **Mono**: 'JetBrains Mono', 'IBM Plex Mono', ui-monospace, Menlo, monospace

## Layout & Posture
- **Radius**: 8px (md) default for cards/buttons.
- **Borders**: 1px hairline borders (`--geist-hairline`).
- **Spacing**: 4px base unit.
- **Tone**: Tech / utility. Data-dense, precise, software-native.
- **Rhythm**: Grid-based, high information density, minimal ornament.
