# Exploration Report: Стадия проекта Expoint ADV

**Date**: 2026-05-12
**Explorer**: AI Explorer (Antigravity)

---

## 1. Problem & Scope

**Core Question**: На какой стадии находится проект Expoint ADV в данный момент?
**Exploration Scope**: Анализ манифестов Genesis v2, PRD, последних логов аудита AntiQ v3, отчетов об улучшениях и недавних задач.

---

## 2. Key Insights

> Проект находится на **стадии архитектурной миграции и подготовки к Production (Genesis v2)**. Первичный SPA-прототип на Vite переводится на Next.js для обеспечения SEO-индексации и серверного рендеринга. 

1. **Технический стек**: Осуществляется переход с Vite на Next.js (App Router), и обновление до Tailwind CSS v4.
2. **Дизайн и UX**: Внедряется концепция "Quiet Luxury" (AntiQ v3). Базовый UI собран, но идет работа над устранением hydration-ошибок и стандартизацией визуального слоя.
3. **Функциональность**: Базовые компоненты (калькулятор, лендинг) сверстаны, но требуют подключения реального бэкенда для AI и CRM.

---

## 3. Detailed Findings

### 3.1 Реализовано (Completed)
- **Базовый UI/UX**: Сверстаны основные секции (Hero, Services, Safety, Benefits, FAQ).
- **Стилизация**: Проведена миграция на Tailwind CSS v4, убраны хардкод-цвета, внедрены семантические токены.
- **Дизайн-система**: Адаптированы принципы "EnterpriseCore" / "Quiet Luxury".
- **Инфраструктура**: Настроен Docker (`docker-compose.yml`) для локального запуска и staging.

### 3.2 В процессе (Work in Progress)
- **Стабилизация Next.js**: Устранение ошибок `use client`, hydration-мисматчей и багов при production-сборке.
- **Улучшение компонентов**: Переход от `Math.random()` к детерминированным данным (seed) для стабильного SSR в ReelStance компонентах.

### 3.3 Ожидает реализации (Pending / Next Steps)
- **3D Визуализатор (WOW-эффект)**: Планируется замена текущего 2D SVG-превью неоновых вывесок на полноценный 3D-движок (Three.js / react-three-fiber).
- **Data Pipeline (CRM)**: Реализация POST-запросов и вебхуков для отправки данных калькулятора (лидов) в Telegram, AmoCRM/Bitrix24 и системы аналитики.
- **Real AI Backend**: Замена моковых задержек (`setTimeout`) в виджете AI-консультанта на реальный API-прокси с использованием `@ai-sdk/google` (Gemini API) и интеграцией базы знаний из NotebookLM.

## 4. Action Recommendations

| Priority | Recommendation | Reason |
|:--------:|----------------|--------|
| **P0** | **Стабилизация Production-сборки Next.js** | Необходима для корректного деплоя и устранения hydration mismatch. |
| **P1** | **CRM Webhooks & Analytics** | Проект должен начать "собирать лиды". Без бэкенда калькулятор работает вхолостую. |
| **P1** | **Интеграция AI Proxy** | Оживление виджета консультанта с помощью реальной LLM. |
| **P2** | **Внедрение Three.js** | Улучшение визуального уровня до Tier-1 Enterprise. |

## 5. Вывод
Проект успешно миновал стадию создания визуального MVP. Текущий этап — **трансформация красивого фронтенда в полнофункциональную, SEO-оптимизированную и конверсионную B2B-платформу**. Главный фокус сейчас смещен с верстки на интеграцию API (Next.js backend), стабилизацию SSR и подключение CRM.
