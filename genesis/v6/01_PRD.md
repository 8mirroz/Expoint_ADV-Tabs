# Product Requirements Document (PRD)
**Project**: Expoint ADV - V6 Multi-Landing SEO & Conversion Architecture
**Version**: 6.0

## 1. Introduction
Переход от единой страницы услуг к масштабируемой мульти-лендинговой архитектуре. Создание отдельных, высококонверсионных, SEO-оптимизированных посадочных страниц для каждой рекламной услуги (объемные буквы, неон, крышные установки и т.д.). Интеграция премиальной GSAP-анимации, контура безопасности и глубокой аналитики.

## 2. Goals & Success Metrics
- **SEO**: Захват органического трафика по узким коммерческим запросам (long-tail B2B).
- **Конверсия (CR)**: Увеличение конверсии в лид на 40% за счет психологического B2B-дизайна и микро-взаимодействий.
- **Производительность**: Google PageSpeed > 90 (Mobile/Desktop) несмотря на GSAP-анимации.
- **Безопасность**: 100% соответствие 152-ФЗ, защита форм от спам-ботов (без визуальной капчи).

## 3. Requirements

### [REQ-001] Мульти-Лендинговая Архитектура
- **Given** систему на Next.js (App Router),
- **When** пользователь переходит на `/services/[slug]`,
- **Then** система должна рендерить уникальный лендинг услуги с динамическими метаданными, специфичными кейсами и калькулятором.

### [REQ-002] SEO и Семантика
- **Given** контентную стратегию,
- **When** поисковой бот сканирует страницу,
- **Then** он должен видеть корректный `next/metadata`, сгенерированные `sitemap.xml`, `robots.txt`, микроразметку Schema.org (Product, LocalBusiness, BreadcrumbList) и семантически верный HTML (H1-H3).

### [REQ-003] Контур Безопасности и Аналитики
- **Given** форму заявки и B2B трафик,
- **When** пользователь отправляет данные или взаимодействует с сайтом,
- **Then** система собирает телеметрию (серверная аналитика, без блокировки UI), проверяет payload (Zod), блокирует ботов (Rate Limiting / Turnstile) и логирует согласие с 152-ФЗ.

### [REQ-004] Premium Motion & GSAP
- **Given** премиальное позиционирование,
- **When** пользователь скроллит страницу,
- **Then** активируются оптимизированные GSAP ScrollTrigger анимации, Lenis smooth scrolling, Parallax эффекты и микро-анимации карточек без просадки FPS.

### [REQ-005] B2B Conversion Psychology
- **Given** архитектуру блока доверия и цен,
- **When** пользователь изучает предложение,
- **Then** контент должен снимать возражения конкурентов (почерпнутые из базы знаний NotebookLM), использовать price anchoring, социальное доказательство (кейсы до/после) и frictionless CTA.

## 4. Design Prompt (Frontend Motion Ultra-Prompt)
*"Разработай компонент React (Next.js) в стиле 'Luxury Industrial B2B'. Используй темную тему (#0a0a0a), градиенты (#00ffcc до #0066ff для акцентов). Интегрируй GSAP ScrollTrigger для staggered fade-up появления секций. Используй Framer Motion для микро-интеракций карточек (hover spring effect, glow stroke). Имплементируй Lenis для smooth scroll. Верстка должна быть pixel-perfect 1280px max-width, с использованием Tailwind CSS v4. Оптимизируй перерисовки (useMemo, React.memo). Компонент должен включать сложные Bento-grid паттерны, типографику Inter/Outfit, и glassmorphism панели (backdrop-blur)."*
