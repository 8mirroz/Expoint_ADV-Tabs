# Genesis v6 - Version Manifest

**Date**: 2026-05-12
**Status**: Active
**Previous**: v5

## Version Goals
Трансформация архитектуры Expoint ADV: создание масштабируемой системы индивидуальных лендингов для каждой услуги с глубокой SEO-оптимизацией, интеграцией контура безопасности и аналитики, а также внедрением премиум-анимаций (GSAP) и UX-паттернов с максимальной конверсией.

## Major Changes
- Мульти-лендинговая архитектура: отдельная посадочная страница для каждой рекламной услуги.
- Встроенная SEO-архитектура: schema.org, SSR-метаданные, семантические ядра страниц.
- Контур безопасности и аналитики (соответствие 152-ФЗ, защита от ботов, телеметрия воронок).
- Premium Motion System (GSAP, Framer Motion) с акцентом на перфоманс.
- AI-UX: адаптация пути пользователя на основе поведенческой психологии и B2B конверсии.
