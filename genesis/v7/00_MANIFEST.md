# Genesis v7 - Version Manifest

**Date**: 2026-05-13
**Status**: Active
**Previous**: v6

## Version Goals
Разработка внутренних страниц платформы Expoint ADV: страница с ценами и тарифами, интерактивный многошаговый калькулятор стоимости, конфигуратор готовых рекламных решений (Product Packs) и система корзины/заказа — с полной интеграцией в существующую инфраструктуру (store, analytics, leads API).

## Major Changes
- Страница `/prices` с тарифными сетками и пакетными предложениями.
- Страница `/calculator` — полноценный конфигуратор с 3D-превью (SignScene).
- Страница `/catalog` — каталог готовых решений (PRODUCT_PACKS) с фильтрацией.
- Корзина (CartStore + Cart UI) с персистентностью и отправкой заявки.
- Конфигуратор пакетов: выбор опций внутри PRODUCT_PACK с живым пересчётом цены.
- Интеграция корзины с `/api/leads` для отправки заказа.
