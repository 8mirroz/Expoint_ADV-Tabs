# Genesis v8 - Version Manifest

**Date**: 2026-05-14  
**Status**: Active  
**Previous**: v7 (Commerce Layer)

## Version Goals
Создать **единую презентационную фреймворк-архитектуру (Presentation Landing Framework)** — систему шаблонов и переиспользуемых блоков, на основе которой генерируются все страницы сайта: главная, услуги, кейсы, каталог, калькулятор, цены, академия, о компании, контакты.

## Major Changes
- [ADD] Единый `PageShell` layout-фреймворк для всех презентационных страниц
- [ADD] Композитный блочный конструктор: `SectionRegistry` для динамической сборки страниц
- [ADD] Типизированная схема страниц `PageBlueprint` для data-driven рендеринга
- [ADD] Полноценные наполненные страницы: About, Academy, Contacts, Portfolio, Cases Archive
- [ADD] Единая навигация с полным маршрутным меню
- [REFACTOR] Header/Footer как часть `PageShell` с вариативностью по типу страницы
- [ADD] Shared section components: HeroGeneric, CTASection, TestimonialsSection, StatsSection
