# Rule: UI/UX Premium Guidelines

**Scope**: All UI components and layouts.

## Aesthetics
- **Industrial Premium**: Clean lines, slate/navy palette, high-contrast typography.
- **Micro-animations**: Smooth transitions (Framer Motion) for all interactions.
- **Glassmorphism**: Subtle use of background-blur for overlays.

## Interaction
- **Feedback**: Every click or action must have an immediate visual response.
- **Graceful Degradation**: Ensure basic functionality works if animations are disabled.
