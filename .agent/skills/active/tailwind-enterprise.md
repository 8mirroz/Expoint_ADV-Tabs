# Skill: Tailwind Enterprise

**Description**: Scalable Tailwind CSS architecture focusing on semantic design tokens and maintainability.

## Rules
- **No Hardcoded Values**: Use CSS variables (tokens) defined in `index.css` (e.g., `bg-[var(--slate-50)]`).
- **Semantic Classes**: Prioritize semantic names for composite components.
- **Config Inheritance**: Extend `tailwind.config.js` with project-specific scales for spacing and shadows.
