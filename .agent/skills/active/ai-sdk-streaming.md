# Skill: AI SDK Streaming

**Description**: Real-time AI integration using Vercel AI SDK and streaming patterns.

## Implementation
- **useChat Hook**: For managing chat state and streaming tokens.
- **Context Injection**: Pattern for sending UI state (e.g., current configurator JSON) to the AI as invisible context.
- **Streaming UI**: Using React Server Components or client-side fragment updates for a "living" response.
