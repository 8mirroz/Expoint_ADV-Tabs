# Skill: React TypeScript Pro

**Description**: Advanced React development with strict TypeScript, emphasizing the Container/Presenter pattern and type-safe components.

## Patterns
- **Container/Presenter**: Decouple logic (data fetching, state) from presentation (UI).
- **Discriminated Unions**: For robust component props (e.g., Alert variants).
- **Generic Components**: Create reusable, type-safe UI components (List, Table).

## Guidelines
- Avoid `any`.
- Use interfaces for external data models, types for component props.
- Prefer `FC` (FunctionComponent) for standard components.
